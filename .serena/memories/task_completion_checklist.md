# タスク完了時のチェックリスト

## 必須チェック項目

1. **テスト実行**

   ```bash
   ./vendor/bin/phpunit
   ```

   - 全テストが通ることを確認

2. **コード品質確認**
   - PHPの構文エラーがないことを確認
   - PSR-4準拠の名前空間とクラス配置

3. **CLAUDE.mdのルール遵守**
   - テストコードがセットで書かれているか
   - ファイル末尾に改行が入っているか

## 推奨チェック項目

- コードスタイルの一貫性
- 適切なdocstring/コメント
- Value Objectsの不変性
- CTL命題のcallableインターフェース遵守
